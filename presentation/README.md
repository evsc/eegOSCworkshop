## Brainwaves

| WAVE | FREQ | STATE |
| ------- | ------- | ------- |
| **GAMMA** | 30-120 Hz | hyper active |
| **BETA** | 14-30 Hz | active thinking |
| **ALPHA** | 7-14 Hz | relaxed focus, meditation |
| **THETA** | 4-7 Hz | drowsy, dreams |
| **DELTA** | < 4 Hz | dreamless sleep |

 3. Amplifier
