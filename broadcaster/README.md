﻿﻿﻿﻿﻿﻿﻿﻿﻿# BroadcastersThe *eeg\_broadcast\_3* processing app receives Muse data via _MuseIO_ and _Muse Direct_. It acts as an OSC broadcasting server. Multiple clients can connect to it to receive simplified Muse data via OSC. <img src="https://raw.githubusercontent.com/evsc/eegOSCworkshop/master/broadcaster/eeg_broadcast_3.png"/>## older versions- *eeg\_broadcast\_2* … receives from muse-io and muse monitor (phone app)- *eeg\_broadcast\_1* … receives from thinkgear, zeo and muse-io
